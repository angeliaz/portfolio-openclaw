# 无风险/低风险套利 SOP

> **定位**：挖掘市场定价失效产生的低风险收益机会。与个股研究不同，套利不依赖对公司基本面的判断，而是依赖**价格关系的收敛**或**结构性规则的执行**。
>
> 用法: `/arbitrage` 或 `/arbitrage [类型]`，例如 `/arbitrage AH溢价`、`/arbitrage 可转债`

---

## ⚠️ 风险等级声明

> 本 SOP 称"无风险/低风险"是相对概念。每类策略都有其特定风险，**不存在绝对无风险的套利**。

| 类型 | 风险等级 | 主要风险来源 |
|------|---------|------------|
| ETF折溢价申赎套利 | ⭐ 极低 | 流动性风险、操作时间差 |
| 可转债转股套利（溢价率<0） | ⭐ 极低 | T+1延迟、转股价修正风险 |
| 打新申购（A股） | ⭐⭐ 低 | 破发风险（新股上市首日跌破发行价） |
| 并购套利 | ⭐⭐⭐ 中 | 交易失败风险、监管否决 |
| AH溢价收敛交易 | ⭐⭐⭐⭐ 中高 | 价差可持续扩大、汇率风险、无法同时平仓 |

## 类型一：AH股溢价套利（收敛交易）

> **逻辑**：同一公司A股和H股本质相同，长期应趋于合理价差。极端溢价时，A股相对H股贵，买H卖A（或等待A回归）。
>
> ⚠️ **注意**：无法完美对冲（A/H两地流动性和资金属性不同），属于收敛交易而非严格无风险套利。

### 扫描方法

```
# 获取所有A股行情（含AH股代码）
mcp__stock-sdk__get_a_share_quotes  codes=["sh601318","sh601398","sh600519"...]

# 获取对应港股行情
mcp__stock-sdk__get_hk_quotes  codes=["02318","01398","03988"...]

# AH溢价率 = (A股价格 / 汇率 / H股价格 - 1) × 100%
# 当前汇率（港币兑人民币）
mcp__tavily__tavily-search: "港元兑人民币汇率 今日"
```

**AH溢价率判断标准：**

| AH溢价率 | 信号 | 策略 |
|---------|------|------|
| > 50% | A股极度高溢价 | 如需在两地配置，优先港股；等待A股回归 |
| 20%–50% | A股明显高溢价 | 港股性价比更高 |
| 0%–20% | 正常溢价范围 | 无明显套利机会 |
| < 0%（H股溢价） | 反常，H股贵于A股 | 罕见，检查原因（通常是流动性/预期差异） |

|---------|---------|------|---------|
| sh601318 | 02318 | 中国平安 | A通常溢价 |
| sh601398 | 01398 | 工商银行 | A通常溢价 |
| sh03988 | 03988 | 中国银行 | — |
| sh600941 | 00941 | 中国移动 | — |
| sh601899 | 01899 | 紫金矿业 | — |

```
mcp__tavily__tavily-search: "AH股溢价率排行 今日"
```

**操作框架（单边）：**

```
AH溢价率 > 40% 且持续3个月：
  → 同一公司，优先配置港股（性价比更高）
  → 不建议同时做空A股+做多H股（执行难，监管限制）

AH溢价率 > 60%（历史极端水平）：
  → 若本已持有A股该标的，考虑是否换仓港股
  → 等待催化剂（沪港通额度、汇率变化）触发回归
```

## 类型二：ETF折溢价套利

> **逻辑**：ETF在二级市场的价格应等于其成分股净值（NAV）。若出现显著偏差，可通过申购/赎回机制套利。
>
> ✅ **接近真正无风险**：价格偏差可精确量化，通过同日申赎在T日锁定利润。

### 套利条件

```
ETF折溢价率 = (ETF市价 - ETF净值) / ETF净值 × 100%

溢价套利（ETF贵）：
  买入成分股 → 向基金公司申购ETF → 卖出ETF
  前提：总费用（成分股买入+申购费+ETF卖出）< 溢价收益

折价套利（ETF便宜）：
  买入ETF → 向基金公司赎回（换回成分股）→ 卖出成分股
  前提：总费用（ETF买入+赎回费+成分股卖出）< 折价收益
```

### 数据获取

```
# 获取ETF行情（含折溢价率）
mcp__stock-sdk__get_fund_quotes  codes=["510300","159915","510050","512010"]

# ETF净值（需要T日净值）
mcp__tavily__tavily-search: "[ETF名称] 今日净值 折溢价率"
```

**实操门槛：**

| 条件 | 说明 |
|------|------|
| 最低资金 | 通常100万以上（需买齐成分股最小份额） |
| 操作时间 | 须在交易日结束前完成申购/赎回提交 |
| 成本估算 | 折溢价率需 > 0.3%（税费+冲击成本）才有利润空间 |
| 适合品种 | 成分股少、流动性好的ETF（如50ETF、300ETF） |

## 类型三：可转债转股套利（负溢价套利）

> **逻辑**：当可转债转股溢价率 < 0 时，意味着按市价转股可以获得高于可转债本身价值的股票，卖出套利。
>
> ✅ **极低风险**：价差锁定，唯一风险是T+1延迟导致股价变化。

### 套利条件

```
转股价值 = 正股价格 / 转股价 × 100
转股溢价率 = (可转债价格 / 转股价值 - 1) × 100%

当转股溢价率 < -1%（扣除手续费后仍有利润）：
  → 买入可转债 → 申请转股（T+1到账） → 卖出正股
```

### 数据获取

```
# 搜索当前负溢价可转债
mcp__tavily__tavily-search: "可转债 负溢价 今日 转股溢价率最低"

# 获取正股行情
mcp__stock-sdk__get_a_share_quotes  codes=[正股代码]

# 转债行情
mcp__stock-sdk__get_a_share_quotes  codes=[转债代码]
```

**关键参数：**

| 参数 | 说明 |
|------|------|
| 转股溢价率 | < -1% 才考虑（要覆盖双边手续费约0.3%） |
| 转股期限制 | 确认该转债处于可转股期内（通常上市6个月后） |
| 强赎风险 | 若公司已触发强赎条件，时间窗口有限 |
| T+1风险 | 转股后T+1才能卖出，留一天价格波动敞口 |

**操作模板：**

```
标的可转债：______  代码：______
当前转债价格：______  转股价：______
正股当前价格：______
转股价值：______ = 正股价格 / 转股价 × 100
转股溢价率：______%（需 < -1%）

预计套利收益：(1 - 转股溢价率) × 投入资金 - 手续费
执行条件：□ 处于转股期  □ 溢价率 < -1%  □ 正股流动性充足
风险敞口：T+1期间正股波动（已买入转债→转股→等待1日）
```

## 类型四：A股打新策略

> **逻辑**：A股 IPO 定价受监管管控，历史上新股上市首日涨幅显著。打新本质是免费期权——中签才需要资金，不中签不占用资本。
>
> ⭐⭐ 低风险，但**近年破发概率上升**，需选择性参与。

### 打新策略框架

**参与条件（全部满足才参与）：**

| 条件 | 判断标准 |
|------|---------|
| 发行PE不超行业平均 | 发行PE < 同行业A股平均PE × 1.2 |
| 公司质量基本面 | 三年营收增速 > 10%，净利润为正 |
| 市场温度 | A股市场情绪不处于极度低迷 |
| 行业热度 | 行业热度不处于🔴过热（破发概率高） |

**可转债打新（债券打新）：**

```
# 查询近期可转债发行
mcp__tavily__tavily-search: "可转债 申购 本周 打新"
```

- 信用申购，无需资金锁定
- 中签后持有 or 上市首日卖出
- 溢价率通常0-30%，破发风险低于股票打新

**跟踪来源：**

```
mcp__tavily__tavily-search: "A股新股申购 本周 打新日历"
mcp__tavily__tavily-search: "可转债申购 本周 打新"
```

## 类型五：并购套利（事件驱动）

> **逻辑**：并购公告后，目标公司股价通常低于收购价（因为交易存在失败风险）。买入目标公司股票，等待交易完成获得收购溢价。
>
> ⭐⭐⭐ 中等风险。**核心风险**：交易失败（监管否决、买方撤回、条件未达成）。

### 筛选标准

| 条件 | 要求 |
|------|------|
| 收购溢价空间 | 当前股价 < 收购价 × 95%（至少5%价差） |
| 交易类型 | 现金要约收购（vs 换股，后者更复杂） |
| 监管障碍 | 反垄断风险低（非行业龙头，跨境并购需额外评估） |
| 时间成本 | 预期完成时间 ≤ 12个月（年化收益率合理） |

### 数据获取

```
mcp__tavily__tavily-search: "A股 要约收购 公告 最新"
mcp__tavily__tavily-search: "港股 私有化 要约 最新"
mcp__tavily__tavily-search: "美股 merger arbitrage 最新"
```
