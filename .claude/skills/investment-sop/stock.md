## 一、投资哲学层级声明

### 1.1 核心纪律（不可违背）

| 纪律 | 一句话 | 在本SOP中的落地规则 |
|------|--------|-------------------|
| **Never lose money** | 避免永久性资本损失，短期浮亏可忍 | 研究第4步「否决门」任一红线命中→直接否决 |
| **不懂不做** | 解释不清/验证不了/写不出失败路径，就不要下注 | 第4步能力圈红线；Pre-mortem写不出≥3条路径→否决 |
| **Do the right things, do things right** | 优先选对公司，再考虑价格 | 先定性（第1步）再定量（第2-3步）；先确认质量再谈估值 |
| **永远只投入可用金融资产** | 只用即使回撤50%也不影响生活的钱 | 本SOP对"可用金融资产"有效，不含应急金 |

## 研究流程（四步）

```
研究第1步：定性分析
  获取实时行情 → 业务结构拆解
  → 行业与竞争格局分析（波特五力/竞争地位/进入壁垒）
  → 护城河深研（逐类型拆解 → 过五关）
  → 林奇分类
  → 治理核查A（通用）+ B（港股/中概附加）
        ↓
研究第2步：财报体检
  财报时效确认 → 五张卡（增长/盈利/现金/负债/前瞻）→ 交叉验证第1步定性结论
        ↓
研究第3步：估值分析
  DCF三情景两阶段（核心）→ 辅助指标验证 → 安全边际判断
        ↓
研究第4步：否决门
  芒格逆向思维 + 快速红线 + 取数后红线 + Pre-mortem
  任一命中 → 研究终止（否决）
        ↓
研究结论摘要
  → 保存到 output/ → 供「投资组合决策SOP.md」调用
```

### 1.2 行业与竞争格局分析

> 目的：判断公司所在行业的吸引力，以及公司在行业中的竞争地位是否可持续。

### 1.3 护城河深研

> 核心问题：**给竞争对手100亿，能完全复制这家公司吗？**
>
> 先基于 1.2 行业分析，再逐类型评估公司个体护城河，最后过五关。

#### 第一步：逐类型分析

**品牌护城河**

| 分析问题 | 你的判断与依据 |
|---------|--------------|
| 同等功能产品，消费者愿意为该品牌多付多少？（估算溢价幅度） | |
| 品牌复购率/NPS/用户忠诚度数据如何？ | |
| 竞争对手需要多少年+多少资金才能建立同等品牌认知？ | |
| **强度判断** | □ 强（溢价>15%，建立周期>10年）  □ 中等  □ 弱/不存在 |

**网络效应护城河**

| 分析问题 | 你的判断与依据 |
|---------|--------------|
| 网络效应类型：□ 直接（同类用户互联）  □ 间接（平台两侧） | |
| 当前用户规模是否已过临界点（新用户自然涌入）？渗透率约___% | |
| **关键测试**：竞争对手能否绕过此平台直达目标用户？ | □ 不能  □ 能，但困难  □ 能 |
| **强度判断** | □ 强（绕不过）  □ 中等（能绕但有损失）  □ 弱/不存在 |

**规模优势护城河**

| 分析问题 | 你的判断与依据 |
|---------|--------------|
| 规模每扩大一倍，单位成本大约下降多少？ | |
| 固定成本占总成本比例高吗？边际成本是否趋近于零？ | |
| 当前公司规模是否已远超竞争对手，形成难以追平的成本差？ | |
| **强度判断** | □ 强（成本领先>20%）  □ 中等  □ 弱/不存在 |

**转换成本护城河**

| 分析问题 | 你的判断与依据 |
|---------|--------------|
| 用户切换竞品的直接成本（时间/金钱/数据迁移/培训）有多高？ | |
| 产品是否深度嵌入客户工作流、系统或生态？ | |
| 平均合同周期多长？历史续约率约___% | |
| **强度判断** | □ 强（客户宁愿接受涨价也不切换）  □ 中等  □ 弱/不存在 |

**渠道控制/分发力护城河**

| 分析问题 | 你的判断与依据 |
|---------|--------------|
| 公司是否掌控核心分发入口（线下终端/线上流量/政府采购/专业渠道）？ | |
| 竞争对手是否必须经过此渠道才能触达目标用户？ | |
| 渠道掌控能力是否构成新玩家的进入壁垒？ | |
| **强度判断** | □ 强（竞争对手绕不过）  □ 中等  □ 弱/不存在 |

#### 第三步：过五关综合判断

> 综合 1.1（业务结构）、1.2（行业格局）、以上护城河分析，做最终定性判断。

| 关卡 | 核心问题 | 结论 |
|------|---------|------|
| **商业模式关** | 靠什么赚钱？10年后还能这样赚吗？收入可预测/可复制/可扩展？ | ✅/⚠️/❌ |
| **护城河关** | 护城河是否真实存在且持久？（来自以上逐类型分析） | ✅/⚠️/❌ |
| **管理层关** | 是否诚信？资本配置能力如何？（与 1.5 治理核查联动） | ✅/⚠️/❌ |
| **行业结构关** | 行业是否具有吸引力？（来自 1.2 波特五力结论） | ✅/⚠️/❌ |
| **长期需求关** | 20年后人们还需要这种产品吗？ | ✅/⚠️/❌ |

→ **定性分析结论**：___个✅ ___个⚠️ ___个❌
- 5个✅ → 高质量标的，继续
- 3-4个✅（无❌）→ 中等质量，继续（估值预期保守）
- 任一❌ → 定性不过关，暂停或直接否决

### 1.4 林奇分类

> 巴菲特质量门通过后才用。用途：①选估值指标 ②确定卖出纪律 ③确定仓位类型。

| 类型 | 特征 | 估值指标 | 卖出策略 |
|------|------|---------|---------|
| **缓慢增长** | 增速≈GDP（2-4%），高分红 | 股息率（主）、PE-TTM | 股息率降低 |
| **稳定增长** | 大公司，增速10-12%，抗周期 | DCF（核心）、PE历史分位、PEG | 护城河永久破坏/管理层失信/估值高于DCF乐观30% |
| **快速增长** | 中小公司，增速20-25%，扩张期 | PEG（主）、PS、PE前瞻 | 确认"增速下台阶"信号→立刻清仓/大幅减仓 |
| **困境反转** | 曾经好公司遭遇重大打击 | EV/EBITDA、现金消耗率、恢复后远期PE | 股价恢复到困境前合理估值附近 |
| **周期型** ⛔ | 盈利随行业供需大幅波动 | — | **本体系不适用** |
| **隐蔽资产** ⛔ | 拥有被市场忽略的值钱资产 | — | **本体系不适用** |

**快速增长型"增速下台阶"信号**（满足任一条→触发清仓/大幅减仓）：
- 核心增长指标同比增速连续2个季度下滑，且公司指引/一致预期同步下修
- 增速下滑伴随结构性恶化：毛利率/经营利润率/FCF质量走弱
- 竞争格局逆转：市场份额持续流失、核心产品被替代

**⚠️ 林奇分类前瞻验证（分类完成后必做）**

> TTM高增速≠可以归入快速增长型。必须检验**前向数据**是否支撑分类。

| 检验项 | 方法 | 若不支撑 |
|--------|------|---------|
| 公司全年指引隐含H2增速 | H2隐含值 = 全年指引 - H1实际；计算YoY | H2隐含增速<20% → 分类为稳定增长型或标注"分类存疑" |
| 分析师一致预期增速（下年） | 获取next year consensus | <20% → 重新分类 |
| 管理层措辞变化 | 对比最近两次业绩会展望措辞 | 由乐观转保守 → 触发⚠️ |

**结论**：若前瞻验证不支撑，将林奇分类降一档，估值工具相应调整。

| **政策优先度** | 是否曾被要求做损害股东利益的事？ | 所在行业被定义为"公共服务" | 折现率+0.5%；护城河降一档 |

> **综合判断**：
> - 仓位修正系数 = A核查命中项中**取最低值**（非相乘）
> - 兜底规则：A+B合计命中≥3条 → 总仓位上限不超过正常的50%

### 2.1 五张体检卡

> **第1步结论 → 第2步重点对照**
>
> | 定性结论 | 重点体检卡 |
> |---------|----------|
> | 商业模式可预测/可复制 | 卡一（营收增速稳定） |
> | 护城河：品牌/定价权 | 卡二（毛利率稳定或上升） |
> | 护城河：规模优势 | 卡二（OPM随规模提升） |
> | 管理层资本配置强 | 卡二（ROE长期>15%）+卡三（FCF流向） |
> | 林奇：快速增长型 | 卡一（营收>20%）+卡三（FCF为正） |
> | 林奇：困境反转型 | 卡三（现金覆盖≥18个月）+卡四（负债<4倍EBITDA） |

**卡一：增长能力**

| 指标 | 健康标准 | 危险信号 |
|------|---------|---------|
| 营收增速（YoY） | 稳定增长型>10%；快速增长型>20% | 连续2个季度增速下滑 |
| 归母净利润增速 | 正增长且稳定 | 连续下滑或由盈转亏 |
| 核心业务营收占比 | >70%且稳定 | 主业萎缩，靠非经常性收入撑 |

**卡二：盈利质量**

| 指标 | 健康标准 | 危险信号 |
|------|---------|---------|
| 毛利率 | 稳定或上升 | 连续下降→定价权丧失或成本失控 |
| 经营利润率（OPM） | >10%；规模扩大后应提升 | 转负或大幅下滑 |
| ROE | 长期>15%且稳定 | 持续下降或靠高杠杆制造假高ROE |

> **ROE进阶**：ROE = 净利率 × 周转率 × 杠杆倍数。ROE高主要靠杠杆不算真本事。轻资产看ROE，重资产看ROIC（>资本成本8-10%=在创造价值）。

> ⚠️ **内容/游戏行业特殊注意：毛利率虚高陷阱**
> 游戏/内容公司研发费用走费用表（费用化），不计入成本，导致毛利率虚高（可达60-80%）。
> - **正确做法**：以 OPM（经营利润率，已含研发）为核心判断指标，不要用毛利率评价护城河强度
> - **研发费用率是真实信号**：研发费率骤降 → 未来产品管线枯竭，比毛利率下降更危险
> - **验证方式**：毛利率持续上升 + OPM同步上升 = 真正的盈利改善；毛利率上升但OPM平稳 = 研发在加速消耗利润

**卡三：现金能力**

| 指标 | 健康标准 | 危险信号 |
|------|---------|---------|
| 经营现金流 | 持续为正，且>净利润 | 利润高但现金流差→"纸面利润" |
| 自由现金流（FCF） | 持续为正；FCF用于回购/分红=资本配置佐证 | 持续为负→需不断融资才能活 |
| 账上现金储备 | 覆盖12个月以上运营 | 现金快速消耗+债务高企 |

**卡四：负债健康**

| 指标 | 健康标准 | 危险信号 |
|------|---------|---------|
| 资产负债率 | <60% | >70%且持续上升 |
| 有息负债/EBITDA | <3倍 | >5倍→杠杆过高 |

**卡五：前瞻信号**

> ⚠️ 用户/商户运营数据仅适用于平台/消费类公司；制造/工业类跳过，重点看费用率。

| 指标 | 健康标准 | 危险信号 |
|------|---------|---------|
| 销售费用率变化 | 稳定或下降 | 突然飙升→竞争加剧或用户流失 |
| 研发费用率 | 维持合理水平（5-15%） | 骤降→未来失去技术竞争力 |
| 用户/商户核心运营数据（仅平台类） | 稳定增长 | 核心运营指标下滑比利润下滑更危险 |

**财报体检汇总模板**：

## 研究第3步：估值分析——"这个价格合理吗？"

> 以**DCF为核心**，辅助指标验证。目的是**排除荒谬定价**，给出合理买入价区间。

### 3.1 按林奇分类选估值工具

| 林奇分类 | 主估值指标 | 辅助指标 | 绝对不要用 |
|---------|-----------|---------|-----------|
| 缓慢增长 | 股息率 | PE-TTM、PB | — |
| 稳定增长 | **DCF（核心）** + PE历史分位 | PEG、FCF收益率、Comps | — |
| 快速增长 | PEG | PS、PE前瞻、EV/ARR（SaaS） | 静态PE |
| 困境反转 | EV/EBITDA + 现金消耗率 | PS、恢复后远期PE | 当前PE |

**特殊情景追加方法**（叠加在上方主方法之上）：

| 触发条件 | 追加方法 |
|---------|---------|
| 公司有2+差异显著的独立业务板块 | **SOTP 分部加总**（3.5节） |
| 高估值成长股，DCF假设存在争议 | **逆向DCF**（3.4.1节） |
| 需要横向验证，或行业同类可比 | **Comps 相对估值**（3.6节） |
| SaaS/订阅制科技公司 | **SaaS专项指标**（3.7节） |

**PE使用三大陷阱**：

| 陷阱 | 后果 |
|------|------|
| 周期股陷阱 | 利润高点PE最低，看似便宜实际是顶部 |
| 亏损股陷阱 | PE为负，无法使用 |
| 一次性收益陷阱 | 靠卖资产让利润暴增→用扣非归母净利润算PE |

### 3.4 DCF模板（三情景两阶段）

```
输入层
├── 当前标准化FCF（剔除一次性项目）[工具获取/用户提供]

└── 当前股价 vs 三情景区间
     □ 低于悲观情景 → 安全边际充足

     □ 接近基准情景 → 合理，仅护城河极强标的考虑
```

**增长率输入建议**：

| 公司类型 | 第一阶段基准（年1-5） | 终值 |
|---------|-------------------|------|
| 稳定增长型 | 分析师一致预期×0.8 | 3% |
| 快速增长型 | 分析师一致预期×0.7 | 2.5% |
| 困境反转型 | 保守假设恢复至历史正常FCF | 2% |
| 中国公司额外折扣 | 上述结果再×0.9 | 同上 |

**⚠️ FCF0周期化规则（内容/游戏/资源类公司必做）**

> 用当年指引或单年高点FCF作为基准是DCF最常见的乐观偏差来源。

| 行业特征 | FCF0处理规则 |
|---------|------------|
| 稳定可预测（消费/SaaS） | 使用当前TTM标准化FCF（剔除一次性项目） |
| 内容/游戏（hit-driven） | **使用近3年平均FCF**，不得使用当年指引 |
| 资源/大宗商品（周期性） | 使用中周期均价对应的正常化利润 |
| 困境反转型 | 使用历史正常年份FCF均值 |

> 游戏公司示例：
> - 当年指引利润20亿 → 不可直接用
> - 近3年FCF：12亿/15亿/8亿 → FCF0 = 11.7亿（均值）
> - 差异可导致DCF估值高估30-40%

### 3.4.1 逆向DCF（Reverse DCF）

> 不从0开始算内在价值，而是**从当前股价反推隐含增长率**，再判断这个增长率是否合理。
>
> **核心问题**：市场已经定价了什么？我是否比市场更乐观或更悲观？

**操作步骤**：
1. 固定折现率和终值增长率（与正向DCF一致）
2. 将股价作为目标，反解第一阶段年化增长率 g
3. 将 g 与以下基准对比，得出判断

| 反推增长率 g vs 现实基准 | 结论 |
|----------------------|------|
| g < 分析师一致预期 | 市场低估了增长，可能低估 |
| g ≈ 分析师一致预期 | 定价合理，需额外安全边际才值得买 |
| g > 分析师预期 × 1.5 | 市场过于乐观，当前价格已透支增长 |
| g > 公司历史最高增速 | 股价定价了"完美情景"，风险极高 |

**适用场景**：高估值成长股（AI/科技）；DCF基准情景假设存在争议；快速验证估值合理性。

### 3.5 分部加总估值（SOTP）

> 适用：**1.1 业务结构拆解发现2个以上估值逻辑差异显著的独立板块**（如电商+云+投资，游戏+广告+金融）。

**步骤**：识别可独立估值的板块 → 各板块选估值方法 → 加总 → 减净债务

| 板块 | 营收/利润贡献 | 估值方法 | 参考倍数来源 | 估算价值 |
|------|------------|---------|------------|---------|
| 板块A（核心/成熟业务） | | DCF / EV/EBITDA | 行业Comps | ___亿 |
| 板块B（成长业务） | | PS / EV/ARR | 同类成长股 | ___亿 |
| 板块C（投资/金融资产） | | NAV / 持仓市值折扣 | 持仓公开市值 | ___亿 |
| 公司层面净债务（-）/ 现金（+） | — | — | — | ±___亿 |
| **合计 SOTP 价值** | | | | **___亿** |

→ SOTP每股价值：___元  vs  当前股价：___元  **溢价/折价：___%**

> ⚠️ **集团折价（Conglomerate Discount）**：多元化公司通常以SOTP 20-30% 折价交易，反映市场对跨领域资本配置效率的担忧。判断折价是否合理：管理层历史资本配置记录是否出色？各板块是否有协同效应？

### 3.7 科技/SaaS专项指标

> 适用于SaaS订阅制、平台型科技公司。传统PE/PB对这类公司失效。

**运营质量指标（先看这些，再看估值倍数）**：

| 指标 | 计算方式 | 健康标准 | 危险信号 |
|------|---------|---------|---------|
| **ARR增速** | YoY ARR增长率 | >30%（早期）；>20%（成熟期） | 连续下滑 |
| **NRR（净收入留存率）** | 期末现有客户ARR / 期初ARR | >120% 优秀；>100% 健康 | <100% = 老客户价值在流失 |
| **Rule of 40** | 营收增速% + 经营利润率% | ≥40 健康 | <20 增长换不来利润 |
| **LTV/CAC** | 客户终身价值 / 获客成本 | >3 | <2 获客太贵，难以规模化 |
| **CAC回收期** | CAC / 月均毛利 | <18个月 | >24个月 = 资金效率低 |

> ⚠️ **SaaS估值陷阱**：Rule of 40 达标但 NRR<100% → 靠新客掩盖老客户流失，不可持续。**优先看NRR，其次看Rule of 40。**

**估值倍数参考（EV/ARR）**：

| NRR水平 | 营收增速 | 合理 EV/ARR 参考区间 |
|--------|---------|-------------------|
| >130% | >40% | 15-25x |
| 110-130% | 20-40% | 8-15x |
| 100-110% | 10-20% | 4-8x |
| <100% | <10% | <4x（高度谨慎） |

### 4.2 永久性损失防线
