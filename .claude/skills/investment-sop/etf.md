# ETF 深度研究 SOP

> **版本：v1.0**
>
> ETF研究与个股研究的核心区别：**不做定性分析（无管理层/护城河/否决门）**，聚焦于跟踪指数的估值水平、折溢价、流动性，产出"当前是否值得买入/加仓"的判断。
>
> 用法: `/etf [代码]`，例如 `/etf 510300`、`/etf 159915`、`/etf QQQ`

---

## 研究流程（三步）

```
第1步：基本信息 & 实时行情
  获取行情（折溢价/PE/PB/规模）→ 基金基本面确认
        ↓
第2步：跟踪指数分析
  成分股结构 → 估值水平（PE/PB历史分位）→ 行业/风格偏向
        ↓
第3步：买入判断
  估值分位 + 折溢价 + 技术面 → 结论（买/等/不买）
```

## 第2步：跟踪指数分析

**数据来源：**
```
mcp__tavily__tavily-search: "[ETF名/指数名] 前十大持仓 行业分布"
mcp__tavily__tavily-search: "[指数名] PE历史分位 估值"
```

### 2.1 成分股结构

| 分析项 | 内容 |
|--------|------|
| 指数类型 | □ 宽基（沪深300/中证500/纳指）  □ 行业（消费/科技/医药）  □ 主题  □ 跨境 |
| 前十大持仓 | [列出，及合计占比____%] |
| 行业分布 | [前3大行业及占比] |
| 集中度 | □ 高（前10占>50%）  □ 中  □ 低（分散） |
| 市值风格 | □ 大盘  □ 中盘  □ 小盘  □ 混合 |

→ **结构结论**：ETF实际暴露的风险敞口是______，与预期一致/不一致？

### 2.2 估值水平（核心判断依据）

> 估值历史分位数据来源：理杏仁 / 中证指数官网 / 用户提供

| 指标 | 当前值 | 历史分位（近10年） | 判断 |
|------|--------|-----------------|------|
| PE（TTM） | ___x | ___%分位 | 低估/合理/高估 |
| PB | ___x | ___%分位 | 低估/合理/高估 |
| 股息率 | ___% | — | 参考 |

**估值分位解读**：

| 分位区间 | 含义 | 建议动作 |
|---------|------|---------|
| <20% 分位 | 历史低估区间 | 积极建仓，可适当高仓位 |
| 20-40% 分位 | 偏低估 | 正常建仓 |
| 40-60% 分位 | 历史中位，合理 | 小仓位持有，等待回调 |
| 60-80% 分位 | 偏高估 | 控制仓位，不追高 |
| >80% 分位 | 历史高估区间 | 不买；已持有考虑减仓 |

### 2.3 行业/主题逻辑（行业/主题ETF必做，宽基可跳过）

| 分析项 | 内容 |

| 持有期预期 | 短期主题（<1年）/ 中长期配置（1-3年+） |

## 第3步：买入判断

### 3.1 折溢价分析

> 折溢价 = (ETF价格 - 基金净值) / 基金净值 × 100%

| 折溢价情况 | 操作含义 |
|---------|---------|
| 折价 < -1% | 以低于资产净值买入，有利 |
| 折价/溢价在 ±0.5% 以内 | 正常，对决策无影响 |
| 溢价 > 1% | 多付了超过净值的溢价，不利；等回归或选替代产品 |
| 溢价 > 2% | 强烈不建议买入，等回落 |
