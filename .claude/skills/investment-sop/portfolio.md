# 投资组合决策 SOP

> **对应「投资组合决策SOP.md」v1.0 + 「资产配置框架.md」v3.3**
>
> **定位**：在个股研究完成后，决定是否买入、买多少、怎么管。**依赖当前持仓状态与宏观环境**，时间敏感。
>
> - **输入**：月度快照（`/macro` 产出）+ 个股研究结论摘要（`/stock` 产出）
> - **适用于**：个股买入 + ETF买入 + 持仓再平衡
> - **黄金/对冲/SPY PUT操作** → 查阅「资产配置框架.md」第四章

---

## ⚠️ AI协作边界

| AI做不到的（需要用户提供） | 获取方式 |
|------------------------|---------|
| 当前实际持仓配比 | 富途/券商APP |
| 当前股权层总仓位 | 自行汇总 |
| 各市场实际持仓占比 | 自行汇总 |

| AI能做的 | 说明 |
|---------|------|
| 引用月度快照，计算目标配比 | 基于用户提供的快照结论 |
| 基于研究结论摘要做仓位决策 | 用户提供研究结论，AI出仓位推理 |
| 行业热度折扣计算 | 基于资产配置框架行业热度评级 |

## 工具层级（冲突时谁优先）

```
第0层（生存约束）：达利欧 → 资产配置框架.md（月度快照）
  → 股权层上限 = 宏观象限基准 × 全局温度调节
  → 任何标的仓位不能突破当前股权层上限

第1层（核心框架）：巴菲特
  → 个股研究SOP已完成质量判断
  → 本文件基于研究结论摘要做仓位决策

第2层（否决层）：芒格（已在个股研究SOP第4步执行）
  → 进入本文件 = 已通过否决门

第3层（仓位控制层）：加尔布雷思
  → 市场温度影响仓位上限和入场节奏
  → 不作为个股卖出触发，不改变质量/估值标准
```

## 个股/ETF买入决策（三步）

### 决策第1步：宏观定位——确认股权层上限

> **直接引用当月月度快照，不重新判断。**
> 若快照尚未更新（距上次>1个月）→ 先执行 `/macro` 更新快照，再回来。

**从月度快照中复制填写（不得凭记忆填写）：**

```
引用快照：output/月度快照_____-__.md（____年__月__日）

━━━ 全局层 ━━━

宏观象限：____（增长/下行 × 通胀上升/下降）
象限基准股权范围：____%
全局温度调节：____（极度恐慌+20% / 悲观冷清±0% / 冷静正常±0% / 乐观偏热-10% / 极度狂热-20%）
→ 总股权上限：____%

对冲层：黄金____%  SPY PUT____%

投机层：≤2%
现金层：= 100% − 以上各层合计

━━━ 分市场子上限 ━━━

| 市场 | 温度评级 | 子上限 |
|------|---------|--------|
| A股  | ____    | ≤___% |
| 港股 | ____    | ____  |
| 美股 | ____    | ≤___% |

━━━ 实际配比 vs 目标 ━━━

| 层次 | 目标 | 实际（用户提供） | 偏差 |
|------|------|----------------|------|

| 对冲层 | ___% | ___% | ___% |

| 投机层 | ≤2%  | ___% | ___% |
| 现金层 | 残余 | ___% | ___% |

⚠️ 任一层偏差>5% → 先再平衡，再分析新标的
```

### 决策第2步：行业热度检查

> 引用月度快照「行业热度检测」部分结论，不重新计算。

```
目标标的：______  所属行业：______  市场：A股/港股/美股

行业热度评级（来自月度快照）：
  评级：□ 🔴过热  □ 🟡偏热注意  □ 🟢合理/正常  □ 🔵低温/被忽视
  评级依据（引用快照结论）：______

行业热度折扣系数：
  🔴过热     → ×0.5
  🟡偏热注意  → ×0.8
  🟢合理/正常 → ×1.0
  🔵低温/被忽视→ ×1.0（可适当加大）

→ 本标的行业热度折扣系数：___（0.5 / 0.8 / 1.0）
```

> 若目标行业未在快照中覆盖 → 默认🟢合理，折扣系数=1.0，并注明"行业热度未评级"。

### 决策第3步：仓位决策

> 输入：来自 `/stock` 的研究结论摘要。若研究已过期（>6个月或触发复查事件）→ 先重做研究再执行此步。

**3.1 三维软性评分（从研究结论摘要中提取）**

| 维度 | 评分 | 来源 |
|------|------|------|
| 定性评分 | ✅/⚠️/❌ | 研究结论摘要 |
| 财报评分 | ✅/⚠️/❌ | 研究结论摘要 |
| 估值评分 | ✅/⚠️/❌ | 见下方规则 |

**估值评分规则：**

| 当前股价区间 | 估值评分 |
|------------|---------|
| 低于DCF悲观情景 | ✅ |
| DCF悲观～基准之间 | ⚠️ |
| 接近或高于DCF基准情景 | ❌（除非护城河极强） |
| 高于DCF乐观情景 | ❌ |

**3.2 仓位计算**

```
基础仓位上限（按三维评分）：
  3维全✅ → 可用仓位的20%（重仓上限）
  2维✅   → 可用仓位的10%（小仓位试探）
  1维✅及以下 → 观望，不开仓

× 市场温度修正（来自第1步快照）：
  极度恐慌 → +20%  |  悲观冷清 → +10%  |  冷静正常 → ±0%
  乐观偏热 → −10%  |  极度狂热 → −20%

× 行业热度折扣系数（来自第2步）：
  🔴过热 → ×0.5  |  🟡偏热 → ×0.8  |  🟢合理 → ×1.0  |  🔵低温 → ×1.0

× 治理系数（来自研究结论摘要）：
  ___（1.0=无命中；最低项=___）

→ 综合仓位上限 = 基础 × 市场修正 × 行业折扣 × 治理系数
   最终仓位不得突破第1步确认的市场子上限（取低值）
```

**3.3 仓位类型（按林奇分类）**

| 林奇分类 | 仓位类型 | 说明 |
|---------|---------|------|
| 稳定增长型 | **主力仓位** | 判断简单，容错高，基本面破坏才卖 |
| 快速增长型 | **卫星仓位** | 小仓位，PEG<1时介入，增速下台阶立即清仓 |
| 困境反转型 | **机会仓位** | 仅限曾经好公司暂时遇难，严格小仓位 |
| 缓慢增长型 | **ETF替代** | 高股息ETF比个股更省心 |
| 周期型⛔ | 不碰 | 个股研究SOP已在第1步否决 |

## ETF简化决策流程

> 宽基/行业ETF跳过「个股研究SOP.md」第1-4步，但需执行产品结构红线和估值判断。

### ETF决策三步

```
ETF第1步：宏观定位（与个股相同，引用月度快照）

ETF第2步：产品结构红线（硬门槛，命中任一 → 直接否决）
  □ 杠杆/反向/复杂策略 → 不作为底仓
  □ 合成复制/对手方风险（Swap/复杂衍生品） → 直接否决
  □ 规模/日均成交额/价差不满足最低阈值 → 否决（避免"卖不掉"）
  □ 费率/跟踪误差超出可接受区间 → 否决

ETF第3步：估值分析（按指数类型选对指标）
```

### ETF估值指标速查

| 指数类型 | 主估值指标 | 辅助指标 | 典型指数 |
|---------|-----------|---------|---------|
| 大盘宽基 | PE-TTM百分位 | 席勒PE（美股） | 沪深300、标普500、恒生指数 |
| 中小盘宽基 | PE-TTM百分位 | PB百分位 | 中证500、中证1000 |
| 成长型 | PE中位数百分位 | PS百分位 | 创业板指、纳斯达克100 |
| 红利/高股息 | PE-TTM百分位 | **股息率** | 中证红利、上证红利 |
| 金融行业 | **PB百分位** | 股息率 | 中证银行 |
| 消费/医药 | PE-TTM百分位 | — | 中证消费、中证医药 |

### ETF定投与市场温度对照

| 市场温度 | 定投策略 |
|---------|---------|
| 极度恐慌 | 加大定投（平时2倍） |
| 悲观冷清 | 适当增加定投（平时1.5倍） |
| 冷静正常 | 维持正常定投 |
| 乐观偏热 | 减少定投（平时一半），不停止 |
| 极度狂热 | 暂停定投，考虑适当减仓 |
