# 投资组合管理 SOP

定期回顾当前持仓质量，评估整体风险敞口，输出再平衡建议。

---

## 执行步骤

### Step 1: 读取当前持仓

```bash
python .claude/skills/paper-trading/portfolio.py positions
python .claude/skills/paper-trading/portfolio.py pnl
```

同时读取 `data/watchlist.md` 获取关注标的。

### Step 2: 获取所有持仓的实时行情

```
# A股持仓
mcp__stock-sdk__get_a_share_quotes  codes=[...]

# 港股持仓
mcp__stock-sdk__get_hk_quotes  codes=[...]

# 美股持仓
mcp__stock-sdk__get_us_quotes  codes=[...]
```

### Step 3: 组合风险评估

**维度1: 市场分布**
- A股/港股/美股/现金各占比
- 对比宏观框架建议配置（参考 /macro 最新结论）

**维度2: 行业集中度**
- 单一行业占比是否超过 30%（集中风险）
- 科技/消费/金融/制造 各板块平衡

**维度3: 标的集中度**
- 单一标的占比是否超过 20%（个股集中风险）
- 前5大持仓是否超过总仓位 70%

**维度4: 估值与质量检查**
```
mcp__finance-data__get_financial_summary  symbol=<code>  market=<A|HK|US>
```
- 逐一检查持仓 PE/PB/ROE 是否仍在合理区间
- 标记估值过高（红色预警）或逻辑已兑现的标的

**维度5: 盈亏归因**
- 盈利标的：逻辑是否仍成立？继续持有 or 逢高减仓？
- 亏损标的：是估值错判 or 基本面恶化？止损 or 加仓？

### Step 4: 再平衡决策框架

**减仓信号（考虑卖出）:**
- PE历史分位 > 80%，且近期催化剂已兑现
- 基本面出现实质性恶化（ROE下滑 > 5ppt 连续2年）
- 单一标的浮盈超 30%，且仓位超 20%（锁定部分收益）
- 宏观框架要求降低该市场仓位

**加仓信号（考虑买入）:**
- PE历史分位 < 25%，基本面稳定
- 主力资金持续净流入（A股）
- 技术面形成支撑，RSI < 35
- 宏观框架建议超配该市场

**等待信号（持仓不动）:**
- PE在合理区间（25%-65%分位）
- 基本面稳定，催化剂未到
