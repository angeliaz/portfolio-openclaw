# 宏观分析 SOP

> **对应「资产配置框架.md」v3.3 + 「投资组合决策SOP.md」决策第1步**
>
> **产出物**：月度快照（`output/月度快照_YYYY-MM.md`）。这是所有个股/ETF买卖决策的前置输入，执行任何买卖前必须先有有效快照。
>
> - **月度检视** → 执行完整流程，更新快照
> - **季度检视** → 额外更新全局宏观象限 + 行业热度检测
> - **买卖前** → 仅核实快照是否仍有效（距上次快照<1个月），无需重做

---

## 框架定位：两层结构

```
【第一层：总股权上限】
全局宏观象限（慢变量，季度评估）
    → 各资产层基准范围
    ↓
全局市场温度（快变量，每月/每次操作前）
    → 在基准范围内调节总股权上限
    ↓
输出：总股权上限（硬约束）

【第二层：各市场部署约束】
A股 / 港股(H) / 美股(US) 各自独立市场温度
    → 每个市场股权子上限
    ↓
行业热度检测（季度更新）
    → 对特定行业设仓位折扣
```

---

## 第一层：全局宏观象限判断

> 慢变量，每季度评估一次。信号不明确时维持上一次判断。

### 四象限框架

```
                通胀上升
                    │
  商品/黄金          │    股票（增长支撑，通胀压估值）
  通胀保值资产        │    商品/黄金
                    │
经济下行 ───────────┼────────────── 经济增长
                    │
  债券/黄金          │    股票/高质量企业
                    │    （最优象限）
                通胀下降
```

### 各象限 → 资产层基准范围

| 宏观象限 | 特征 | 股权基准 | 对冲层（含黄金） | 现金参考 |
|---------|------|---------|---------------|---------|
| **增长 + 通胀下降**（黄金时期） | GDP加速，CPI回落，利率下行 | 55–75% | 5–10%（黄金5–8%） | 15–35% |
| **增长 + 通胀上升**（经济过热） | GDP强劲，CPI持续上行，央行加息 | 45–65% | 8–12%（黄金7–10%） | 25–45% |
| **下行 + 通胀下降**（衰退/通缩） | GDP放缓，失业上升，通缩风险 | 35–55% | 8–12%（黄金7–10%） | 35–55% |
| **下行 + 通胀上升**（滞胀，最难） | GDP萎缩，通胀顽固，政策两难 | 25–45% | 10–15%（黄金8–10%） | 40–65% |

### 象限判断方法

| 维度 | 核心指标 | 数据来源 |
|------|---------|---------|
| **全球经济增长方向** | 全球PMI、美国GDP、中国PMI、欧元区PMI、全球企业盈利预期 | `mcp__tavily__tavily-search: "全球PMI [月份]"` |
| **全球通胀方向** | 美国CPI/PCE、中国CPI/PPI、大宗商品价格趋势 | `mcp__tavily__tavily-search: "美国CPI [月份]"` |
| **货币政策** | 美联储声明/点阵图、PBOC方向、全球主要央行动作 | `mcp__tavily__tavily-search: "美联储利率 最新"` |
| **辅助确认** | 美债10年期收益率、美元指数DXY | `mcp__tavily__tavily-search: "美债收益率 10年 今日"` |

> **核心逻辑**：美国是全球流动性锚，中国是全球增长重要变量。两者分化时，综合权重判断，并在快照中记录分化情况。

## 第一层：全局市场温度判断

> 快变量，每月/每次操作前判断。对"当前股票资产整体贵不贵"的综合感知。

**判断信号（三类信号同向才算强信号）：**

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **身边信号** | 非投资圈的人主动推荐股票（跨市场普遍） | 专业投资者普遍回避股票 |
| **媒体信号** | 多市场头条充斥"新时代""这次不一样" | 多市场头条充斥"崩盘""史上最差" |
| **全球估值** | A/H/US主要指数PE多数处于历史高分位 | 多数指数PE处于历史低分位 |
| **全球流动性** | 信用利差极窄、杠杆资金高、IPO大量 | 信用利差大幅扩大、基金赎回潮 |
| **避险情绪** | VIX<15 且全球风险偏好极高 | VIX>30 或多市场同步恐慌 |

**全局温度 → 总股权基准调节：**

| 全局温度 | 调节方向 | 行动含义 |
|---------|---------|---------|
| 极度狂热 | 基准 −15% 至 −20% | 大量积累弹药，停止追涨 |
| 乐观偏热 | 基准 −5% 至 −10% | 放慢入场节奏，不追高 |
| 冷静正常 | ±0% | 按基准执行 |
| 悲观冷清 | ±0% | 维持，等待机会 |
| 极度恐慌 | 基准 +10% 至 +20% | 动用弹药，接近股权上限 |

## 第二层：分市场温度判断

> A股、港股、美股各自独立判断，彼此不干扰。

### A股温度判断

```
# 获取A股主要指数估值
mcp__stock-sdk__get_a_share_quotes  codes=["sh000001","sh000300","sz399001","sz399006"]
mcp__stock-sdk__get_industry_spot   (选3-5个重点行业)
mcp__stock-sdk__get_fund_flow       codes=["sh000001"]
```

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 沪深300/上证PE历史80%+分位 | PE历史20%−分位 |
| **情绪/资金** | 两融余额高位/新开户暴增/公募发行火爆 | 两融余额萎缩/基金大额赎回 |
| **成交量** | 单日成交持续超1.5万亿 | 单日成交持续低于7000亿 |
| **政策情绪** | 利好已充分定价、市场快速拉升后高位震荡 | 政策超预期但市场仍低迷 |

### 港股(H)温度判断

```
mcp__stock-sdk__get_hk_quotes  codes=["HSI","HSTECH","HSCEI"]
mcp__tavily__tavily-search: "南向资金 今日净流入"
```

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 恒指/国企指数PE历史80%+分位/AH溢价收窄至低位 | PE历史20%−分位/AH大幅折价 |
| **南向资金** | 南向持续大幅净流入、港股通额度饱和 | 南向持续净流出/外资大幅撤离 |
| **全球风险偏好** | 全球高收益利差极窄/美元走弱 | 高收益利差扩大/美元持续走强 |
| **流动性** | 港股成交量持续放大/杠杆资金高 | 港股成交持续低迷/流动性溢价显著 |

### 美股(US)温度判断

```
mcp__stock-sdk__get_us_quotes  codes=["SPY","QQQ","DIA","IWM"]
mcp__tavily__tavily-search: "VIX指数 今日"
```

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 标普500/纳斯达克PE历史80%+分位/Shiller CAPE高 | PE历史20%−分位 |
| **波动率** | VIX < 13（市场极度自满） | VIX > 30（恐慌） |
| **情绪/资金** | 散户持仓比例极高/AAII看涨极高/IPO大量 | AAII看跌极高/基金大额赎回 |
| **信用市场** | 高收益债利差极窄 | 高收益债利差大幅扩张 |

### 分市场温度 → 子上限

| 该市场温度 | 子上限（占总资产%） | 说明 |
|----------|-----------------|------|
| 极度狂热 | ≤ 10% | 严控，不追涨；已持仓考虑减仓 |
| 乐观偏热 | ≤ 30% | 谨慎，不加仓，好标的小仓试探 |
| 冷静正常 | ≤ 50%（受全局上限约束） | 正常运作 |
| 悲观冷清 | 不设独立子上限 | 机会驱动，可积极建仓 |
| 极度恐慌 | 不设独立子上限 | 动用弹药积极入场 |

## 行业热度检测（季度更新）

> 每季度更新一次。个股买入前的行业级风险过滤层。
> 即使个股质量很好，若整个行业处于泡沫化状态，仍需压缩仓位。

**热度评级 → 仓位折扣系数：**

| 热度评级 | 折扣 | 含义 |
|---------|------|------|
| 🔴 过热/泡沫 | ×0.5 | 该行业个股仓位额外压缩50%，已持仓考虑减仓 |
| 🟡 偏热注意 | ×0.8 | 正常分析，但仓位打八折 |
| 🟢 合理/正常 | ×1.0 | 按个股分析正常配置 |
| 🔵 低温/被忽视 | ×1.0 | 不额外限制，可积极寻找 |

## 注意事项

- 所有数据必须标注来源和日期：`[工具获取, YYYY-MM-DD]` / `[用户提供]`
- 象限信号不明确时，维持上一次判断，不强行调整
- **行业热度 ≠ 市场温度**：市场整体冷清时，特定行业仍可能过热（如A股整体悲观但AI板块极度亢奋）
- **子上限之和通常超过全局股权上限** — 这是正常的；三市场合计不超过全局上限是最终兜底
