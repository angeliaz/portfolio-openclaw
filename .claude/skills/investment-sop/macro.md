# 宏观分析 SOP

覆盖 A/H/US 三市场的宏观经济环境评估，输出资产配置框架建议。

---

## 分析框架（5维度）

### 1. 全球流动性环境
- 美联储政策：利率水平、缩表进度、下次议息预期
- 中国央行政策：LPR、MLF、逆回购利率、降准预期
- 美元指数走势与人民币汇率
- 全球资金流向（新兴市场 vs 发达市场）

### 2. 经济基本面
**中国:**
- GDP增速（季度）、PMI（制造业/非制造业）
- CPI/PPI（通胀/通缩压力）
- 社融/M2（信用扩张速度）
- 出口数据、房地产数据

**美国:**
- GDP增速、非农就业/失业率
- CPI/PCE（核心通胀）
- ISM制造业/服务业PMI
- 企业盈利预期（标普500 EPS）

### 3. 市场估值温度计
| 市场 | 关键指标 | 获取方式 |
|---|---|---|
| A股 | 沪深300 PE/PB、万得全A市盈率 | `mcp__stock-sdk__get_a_share_quotes` codes=["sh000300","sz399001"] |
| 港股 | 恒生指数 PE/PB、恒生科技 PE | `mcp__stock-sdk__get_hk_quotes` codes=["HSI","HSTECH"] |
| 美股 | 标普500 PE、纳斯达克100 PE | `mcp__stock-sdk__get_us_quotes` codes=["SPY","QQQ"] |

估值解读：
- PE分位数 < 30%: 低估区（超配机会）
- PE分位数 30-70%: 合理区（标配）
- PE分位数 > 70%: 高估区（低配/减仓）

### 4. 市场情绪 & 风险偏好
- VIX 恐慌指数水平（>30 恐慌，<15 贪婪）
- A股成交量/换手率（热度指标）
- 北向资金净流向（外资态度）
- 行业轮动速度（频繁 = 投机氛围浓）

**数据获取:**
```
A股行业资金流: mcp__stock-sdk__get_industry_spot (各板块涨跌)
A股概念热点:  mcp__stock-sdk__get_concept_list + get_concept_spot
VIX/情绪:    WebSearch "VIX指数 今日" / "北向资金 今日"
```

### 5. 政策与事件风险
- 中国：政策导向（产业政策、房地产政策、货币政策）
- 美国：美联储声明、财政政策、贸易政策
- 全球：地缘政治风险、大宗商品价格（原油/黄金/铜）
