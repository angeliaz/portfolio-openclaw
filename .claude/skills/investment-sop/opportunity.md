# 投资机会挖掘 SOP

> **定位**：主动扫描市场，系统性发现潜在的值得深度研究的投资机会。本 SOP 输出的是**研究候选清单**，不是买入决策——发现候选后仍需执行 `/stock` 做深度研究，再通过 `/portfolio` 做仓位决策。
>
> 用法: `/opportunity` 或 `/opportunity [方向]`，例如 `/opportunity 困境反转`、`/opportunity 宏观轮动`

---

## 机会挖掘在决策链中的位置

```
宏观快照（/macro）
    → 判断当前适合哪类机会
    ↓
机会挖掘（本文档）
    → 输出候选标的清单（5-10个）
    ↓
个股深度研究（/stock）
    → 对候选标的执行完整四步研究
    ↓
仓位决策（/portfolio）
    → 买多少、怎么管
```

## 五类挖掘方法

---

### 方法一：宏观轮动筛选

> **逻辑**：不同宏观象限有不同受益板块。当宏观象限切换时，先于市场布局下一个受益板块。

**宏观象限 → 受益资产/板块对照表：**

| 宏观象限 | 受益方向 | A股板块 | 港股 | 美股 |
|---------|---------|---------|------|------|
| **增长 + 通胀下降**（黄金时期） | 成长股、科技、消费 | 科技/消费龙头 | 互联网/科技 | 纳斯达克/标普 |
| **增长 + 通胀上升**（经济过热） | 大宗商品、能源、金融 | 银行/资源/周期 | 资源股 | 能源/大宗ETF |
| **下行 + 通胀下降**（衰退/通缩） | 防御性股（高股息）、债券 | 红利/高股息ETF | 公用事业/电信 | SPY/国债 |
| **下行 + 通胀上升**（滞胀） | 黄金、大宗商品 | 贵金属/黄金ETF | 资源股 | GLD/商品ETF |

**操作流程：**

```
第一步：读取最新月度快照，确认当前宏观象限
→ output/月度快照_YYYY-MM.md

第二步：判断象限是否发生切换或即将切换
mcp__tavily__tavily-search: "全球PMI 最新数据 [月份]"
mcp__tavily__tavily-search: "美国CPI 趋势 最新"

第三步：按对照表，锁定受益板块
mcp__stock-sdk__get_industry_list   → 找对应A股行业板块代码
mcp__stock-sdk__get_industry_spot   → 查板块当前估值和涨跌

第四步：板块内筛选龙头（PE历史低分位 + ROE稳定）
mcp__stock-sdk__get_industry_constituents  symbol=<板块代码>
mcp__finance-data__get_financial_summary   symbol=<代码>  market=A
```

**输出：** 3-5个处于象限受益方向且估值合理的候选标的

### 方法二：困境反转扫描

> **逻辑**：曾经的好公司因短期负面事件（政策打压、行业寒冬、管理层变动）股价大幅下跌，但护城河未被永久破坏，未来有恢复可能。这是林奇框架中"困境反转型"的主动挖掘版。

**触发信号（满足任一条）：**

- 股价距历史高点回撤 > 50%，且持续超过12个月
- 行业经历了政策重创或供给冲击，龙头盈利阶段性恶化
- 管理层换届或股权结构调整后，新管理层开始回购/分红
- 行业景气触底回升信号出现（库存去化、价格止跌）

**数据获取：**

```
# A股行业低估板块
mcp__stock-sdk__get_industry_spot   → 找PE历史低分位的板块
mcp__stock-sdk__get_industry_kline  symbol=<代码>  period=week  startDate=<3年前>

# 搜索困境反转主题
mcp__tavily__tavily-search: "A股 困境反转 龙头 估值低 [行业名] [年份]"
mcp__tavily__tavily-search: "港股 超跌龙头 业绩触底 [年份]"
mcp__tavily__tavily-search: "美股 turnaround 低估值 [行业] [年份]"

# 个股层面
mcp__stock-sdk__get_kline_with_indicators
  symbol=<候选代码>
  period=week
  startDate=<3年前>
  indicators=["MA","RSI"]
```

**快速筛选标准（全部满足才进入深度研究）：**

| 标准 | 门槛 |
|------|------|
| 股价回撤 | 距历史高点 > 50% |
| PE历史分位 | < 20%（或处于历史极低区） |
| 公司曾经质量 | 回撤前ROE > 15% 超过3年 |
| 护城河判断 | 行业地位/品牌/技术仍在，非结构性破坏 |
| 现金流 | 现金储备 > 12个月运营，未进入生存危机 |
| 否决门快查 | 无重大欺诈/监管永久性否定 |

**候选清单模板：**

```
| 标的 | 市场 | 当前PE分位 | 距高点回撤% | 困境原因 | 反转逻辑 | 优先级 |
|------|------|-----------|-----------|---------|---------|--------|
```

### 方法三：量化初筛（价值+质量双因子）

> **逻辑**：用量化指标快速过滤，找出估值合理且质量过硬的标的，作为深度研究的输入池。

**核心筛选公式（两种选一）：**

**A. 巴菲特式筛选（质量优先）**

```
筛选条件（全部满足）：
  ROE > 15%（近3年均值）
  净利率 > 10%
  资产负债率 < 60%
  PE-TTM < 历史50%分位（不贵）
  PE/ROE < 1.5（质量与估值匹配）
```

**B. 格雷厄姆式筛选（安全边际优先）**

```
筛选条件：
  PB < 1.2（股价低于净资产）
  PE-TTM < 15x
  近3年ROE > 10%（资产质量尚可）
  流动比率 > 2（财务安全）
```

**C. PEG筛选（成长+估值匹配）**

```
筛选条件：
  PEG = PE / 年化营收增速 < 0.8
  营收增速 > 15%（近3年均值）
  净利润增速 > 10%
  FCF为正
```

**数据获取：**

```
# 行业整体估值
mcp__stock-sdk__get_industry_spot   → 找PB/PE偏低的行业
mcp__stock-sdk__get_industry_constituents  symbol=<板块>

# 个股财务摘要（批量查看）
mcp__finance-data__get_financial_summary  symbol=<代码>  market=A|HK|US

# 搜索筛选结果辅助
mcp__tavily__tavily-search: "A股 低PB高ROE 筛选 [行业] [年份]"
mcp__tavily__tavily-search: "港股 低估值 高股息 稳定增长 [年份]"
```

### 方法四：事件驱动机会

> **逻辑**：特定企业事件会短期扭曲股价，产生脱离基本面的错误定价，给有准备的投资者创造机会。

**常见事件类型及策略方向：**

| 事件类型 | 通常影响 | 机会窗口 | 核心问题 |
|---------|---------|---------|---------|
| **管理层换届（创始人回归/优质新CEO）** | 短期不确定，中期可能向好 | 公告后1-3个月 | 新管理层的能力和激励是否到位？ |
| **业绩大幅低于预期（Bad news out）** | 股价暴跌 | 暴跌后1-2周 | 是一次性事件还是结构性恶化？ |
| **监管利好/行业政策松绑** | 板块性机会 | 政策发布后 | 政策力度是否足够？落地节奏？ |
| **大规模回购公告** | 股价支撑，看多信号 | 公告后持续 | 回购是否实质性执行（而非口头承诺）？ |
| **分拆/剥离低效资产** | 释放隐藏价值 | 公告后-完成前 | 分拆后母公司+子公司合计价值 vs 当前市值？ |
| **股价异动（大跌但无实质负面）** | 短期过度反应 | 当天-1周 | 是否有实质基本面变化？ |

**事件监控方法：**

```
# 实时财经事件
mcp__tavily__tavily-search: "A股 重大公告 今日 回购/分拆/管理层"
mcp__tavily__tavily-search: "港股 盈利预警 公告 本周"
mcp__tavily__tavily-search: "美股 earnings beat/miss 本周 重大"

# 政策事件
mcp__tavily__tavily-search: "产业政策 最新 [行业] [月份]"
mcp__tavily__tavily-search: "监管政策 松绑 [行业] 最新"

# 异动个股
mcp__stock-sdk__get_a_share_quotes  codes=[异动股代码]
mcp__stock-sdk__get_today_timeline  symbol=<代码>  # 查看分时量价
```

**事件驱动快速评估框架：**

```
事件：______  发生时间：______  标的：______

①事件性质：□ 一次性（可恢复）  □ 结构性（不可逆）
②股价反应：暴跌___% / 上涨___%  是否过度？______
③基本面变化：□ 无变化  □ 短期影响  □ 长期破坏
④投资机会：□ 超卖后买入  □ 消息兑现后减仓  □ 无机会（等待）
⑤时间窗口：预计____内恢复/兑现
```

### 方法五：跨市场对标比较法

> **逻辑**：同业务模式的公司在不同市场估值不同。找出被低估市场中的优质标的，尤其当一个市场因非基本面原因（情绪、政策、流动性）被系统性压制时。

**常用对标组合：**

| 低估市场标的 | 对标参照 | 关注的价差逻辑 |
|-----------|---------|-------------|
| 腾讯（港股）vs Meta（美股） | 相同广告+生态模式 | 港股/中概长期折价是否合理？ |
| 阿里巴巴（港股/US）vs Amazon | 电商+云 | AI改造预期是否充分定价？ |
| 招商银行（A股）vs 全球优质银行 | ROE > 15% 的银行 | A股银行PB < 1，为什么？ |
| 中国平安 vs 友邦保险 | 保险龙头 | AH溢价 + 地产敞口担忧是否过度？ |
| 中芯国际（港股/A股）vs 台积电 | 代工逻辑 | 技术代差 + 地缘折价多少合理？ |

**操作步骤：**

```
第一步：确认对标逻辑（业务模式高度相似）
第二步：获取两者估值指标
mcp__finance-data__get_financial_summary  symbol=<A>  market=HK
mcp__finance-data__get_financial_summary  symbol=<B>  market=US
mcp__stock-sdk__get_hk_quotes  codes=[港股代码]
mcp__stock-sdk__get_us_quotes  codes=[美股代码]

第三步：量化价差
  PE差距：A公司PE = __  B公司PE = __  差距 = __%
  PB差距：A公司PB = __  B公司PB = __
  ROE差距：A公司ROE = __  B公司ROE = __

第四步：解释价差原因（合理 vs 不合理）
  合理折价因素：政策风险/地缘风险/流动性/审计风险
  不合理部分（情绪折扣）：______

第五步：判断折价是否有望收窄（需要什么催化剂）
```
