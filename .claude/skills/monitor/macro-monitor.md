# 宏观深度监控 (Macro Monitor)

定时（建议每月一次）或手动触发，执行完整宏观分析，更新资产配置框架。
本 skill 调用 `investment-sop/macro.md` 的完整方法论执行，并与上期结论对比变化。

---

## 执行协议

### Phase 1: 数据采集（并行）

**市场温度数据:**
```
mcp__stock-sdk__get_a_share_quotes  codes=["sh000001","sh000300","sz399001","sz399006"]
mcp__stock-sdk__get_hk_quotes       codes=["HSI","HSTECH","HSCEI"]
mcp__stock-sdk__get_us_quotes       codes=["SPY","QQQ","DIA"]
```

**A股行业全景:**
```
mcp__stock-sdk__get_industry_list
mcp__stock-sdk__get_industry_spot  symbol=<主要行业，选10-15个>
mcp__stock-sdk__get_fund_flow       codes=["sh000001","sh000300"]
```

**宏观经济数据（WebSearch，按需选取）:**
```
WebSearch: "中国GDP 最新季度数据"
WebSearch: "中国PMI制造业 非制造业 最新"
WebSearch: "中国CPI PPI 最新月度"
WebSearch: "LPR 央行货币政策 最新"
WebSearch: "社融 M2 最新数据"
WebSearch: "美联储利率决议 最新"
WebSearch: "美国CPI PCE 最新"
WebSearch: "美国非农就业 最新"
WebSearch: "美元指数 人民币汇率 最新"
WebSearch: "VIX恐慌指数 最新"
WebSearch: "北向资金 本月净流向"
```

### Phase 2: 宏观分析（按 macro.md 框架执行）

完整执行 `investment-sop/macro.md` 的5维度分析框架：
1. 全球流动性环境
2. 经济基本面（中国 + 美国）
3. 市场估值温度计
4. 市场情绪 & 风险偏好
5. 政策与事件风险

### Phase 3: 与上期对比

读取上期宏观报告（如存在）：`data/research-notes/macro-YYYY-MM.md`

对比变化：
- 哪些维度评级发生变化（上调/下调）？
- 资产配置建议是否需要调整？
- 新出现的风险点或机会？

### Phase 4: 输出报告

保存至 `data/research-notes/macro-YYYY-MM.md`

---
