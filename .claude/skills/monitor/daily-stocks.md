# 每日个股扫描 (Daily Stocks)

对 `data/watchlist.md` 中的所有标的进行每日快速异动扫描。目标 ≈5分钟完成。

---

## 执行步骤

### Step 1: 读取 watchlist

```
Read: data/watchlist.md
→ 提取所有标的代码，按市场分组
  A股: [...]
  港股: [...]
  美股: [...]
```

### Step 2: 批量获取实时行情（并行调用）

```
mcp__stock-sdk__get_a_share_quotes  codes=[A股代码列表]
mcp__stock-sdk__get_hk_quotes       codes=[港股代码列表]
mcp__stock-sdk__get_us_quotes       codes=[美股代码列表]
```

### Step 3: A股标的资金流向（仅A股）

```
mcp__stock-sdk__get_fund_flow  codes=[A股代码列表]
```

### Step 4: 异动标的深挖

对当日涨跌幅 **≥ ±3%** 的标的，额外执行：
```
WebSearch: "[公司名] 今日 异动 原因"
mcp__stock-sdk__get_panel_large_order  codes=["<异动标的>"]
```

---

## 异动判断标准

| 信号类型 | 条件 | 标注 |
|---|---|---|
| 强势异动 | 单日涨幅 ≥ +5% | 🚀 |
| 上涨信号 | 单日涨幅 +3% ~ +5% | 🟢 |
| 轻微波动 | 单日涨跌 ±3% 以内 | ➖ |
| 下跌信号 | 单日跌幅 -3% ~ -5% | 🔴 |
| 强势下跌 | 单日跌幅 ≤ -5% | ⚠️ |

## 注意
- 重点关注：持仓标的优先于 watchlist 中未持仓标的
- 成交量异常（> 5日均量 150%）需单独标注
- 跌幅 > 5% 的持仓标的需立即分析原因，考虑是否需要操作
