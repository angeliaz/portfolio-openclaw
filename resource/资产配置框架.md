# 资产配置框架

> **版本：v3.3 — 五层全动态版 + A/H/US分市场温度 + 行业产业链分析层**
>
> **核心定位**：个股决策的前置约束层。解决"总资产怎么分配、每类资产配多少"，不解决"买哪只股票"。
>
> - **月度检视时** → 重新判断宏观象限，更新目标配比，记录快照
> - **每次买卖前** → 确认当前股权层可用空间未超上限
> - **对冲层操作时** → 查第四章：黄金时机指引 + SPY PUT纪律规则
> - **不确定该看哪个文档？** → 查阅「投资体系总览.md」

## 一、底层哲学

### 1.1 为什么需要资产配置

个股决策解决"买什么、什么价格"，资产配置解决更上层的问题：

> **你的总资产应该怎么分配？每类资产占多少？什么时候调整？**

研究表明，投资组合90%以上的回报差异来自资产配置，而非选股。选对股票但配置错误，结果仍然糟糕。

### 1.2 本框架哲学：巴菲特式配置

本框架不采用达利欧的"全天候均衡防守"，而是**以高质量股权为核心、配足子弹等待机会的猎手模式**：

| 对比维度 | 达利欧全天候 | 本框架（巴菲特兼容） |
|---------|------------|------------------|
| 核心信念 | 无法预测未来，均匀布防所有场景 | 能深度理解优质企业，长期持有复利增长 |
| 股票仓位 | 30%（风险平价） | 0-95%（全动态，质量+价格驱动） |
| 债券角色 | 40-55%，衰退防御核心 | 0-10%，暂搁置，不理解的资产不配置 |
| 现金角色 | 固定5-10% | 0-95%，残余项，反映"当前没有合理定价的好资产" |
| 黄金角色 | 7.5%，固定 | 0-10%，对冲层一部分，宏观信号驱动 |

> **达利欧的宏观分析框架仍然有价值**——四象限思想用来判断当前宏观环境；但他的具体组合比例与巴菲特股权长持哲学不兼容，不直接照搬。

### 1.3 五层资产定义

| 资产层 | 范围 | 角色 | 核心逻辑 |
|-------|------|------|---------|
| **股权层** | 0%–95% | 财富主引擎 | 优质企业复利是财富积累的核心来源 |
| **对冲层** | 0%–15% | 保险/尾部保护 | 对冲货币风险、系统性风险和极端事件；不以盈利为主要目的；仅含黄金和SPY PUT |
| **债券层** | 0%–10% | 低风险收益补充 | 暂搁置；货币基金已承担其流动性功能 |
| **现金层** | 0%–95% | 战略弹药 | 残余项，不设目标；高现金=当前没有合理定价的好资产；工具为货币基金/短债等稳定品种 |
| **投机层** | 0%–2% | 纯投机 | 无内在价值锚、极高波动；可接受100%亏损；与对冲层风险特征完全不同，独立一层 |

**五个原则：**

1. **每一层最低都可以是0%**，取决于当时的市场和宏观环境
2. **股权层越高，越依赖判断质量**——只有在高置信度的优质标的才能推满
3. **现金是结果，不是目标**——先确定股权+对冲+债券+投机，现金是剩余
4. **五层之和必须等于100%**：现金层 = 100% − 股权 − 对冲 − 债券 − 投机
5. **对冲层 ≠ 投机层**：黄金/SPY PUT是保险（负相关于股权），加密货币是投机（与股权高度正相关）；两者风险特征根本不同，不能混放

## 二、配置决策框架

### 2.1 决策流程

两层结构：第一层决定总股权上限，第二层决定各市场如何部署。

```
【第一层：总股权上限】

输入：全局宏观象限（慢变量，每季度评估一次）
    全球综合视角（美/中/欧经济增长 × 通胀方向）
    → 各资产层基准范围
    ↓

调节：全局市场温度（快变量，每次操作前判断）
    对全球股权整体偏贵还是偏便宜的综合感知
    → 在基准范围内上下调节总股权目标
    ↓

输出：总股权上限（这是全局硬约束）

────────────────────────────────────────

【第二层：各市场部署约束】

A股 / 港股(H) / 美股(US) 各自独立判断市场温度
    → 每个市场设独立子上限
    → 三市场子上限之和通常超过总股权上限（正常，因为不会同时满配三市场）
    ↓

实际部署：完全由个股定性+定量判断驱动
    哪个市场有好标的且合理定价 → 在子上限内买入
    三市场实际持仓合计不超过总股权上限
    ↓

月度快照：记录全局判断 + 三市场温度，下月对比
```

---

### 2.2 全局宏观象限 → 各资产层基准范围

**达利欧核心洞察：未来只有四种可能的经济环境，每种都有表现最好的资产。**

```
                    通胀上升
                       │
     商品 / 黄金        │    股票（增长支撑，通胀压缩估值）
     通胀保值资产       │    商品 / 黄金
                       │
  经济下行 ─────────────┼───────────── 经济增长
                       │
     债券 / 黄金        │    股票 / 高质量企业
                       │    （最优象限）
                    通胀下降
```

**四象限 → 各层基准范围（供参考，非精确公式）：**

| 宏观象限 | 特征 | 股权基准 | 对冲层（含黄金） | 现金参考 |
|---------|------|---------|---------------|---------|
| **增长 + 通胀下降**（黄金时期） | GDP加速，CPI回落，利率下行 | 55–75% | 5–10%（黄金5–8%） | 15–35% |
| **增长 + 通胀上升**（经济过热） | GDP强劲，CPI持续上行，央行加息 | 45–65% | 8–12%（黄金7–10%） | 25–45% |
| **下行 + 通胀下降**（衰退/通缩） | GDP放缓，失业上升，通缩风险 | 35–55% | 8–12%（黄金7–10%） | 35–55% |
| **下行 + 通胀上升**（滞胀，最难） | GDP萎缩，通胀顽固，政策两难 | 25–45% | 10–15%（黄金8–10%） | 40–65% |

> **注意**：表格给出的是参考方向，不是硬性公式。实际配比由你对当前机会质量的判断驱动——找不到合理定价的好标的，股权层可以低于下限；找到了极度低估的好公司群，可以接近上限。

**如何判断当前全局象限（综合三市场信号）：**

| 维度 | 核心指标 | 判断方法 |
|------|---------|---------|
| **全球经济增长方向** | 全球综合PMI、美国GDP趋势、中国PMI、欧元区PMI、全球企业盈利预期 | 多数主要经济体PMI同向 → 方向明确；分化严重 → 维持上期判断 |
| **全球通胀方向** | 美国CPI/PCE、中国CPI/PPI、全球大宗商品价格趋势 | 美国通胀方向为主锚，辅以中国PPI（领先大宗）判断全球通胀走向 |
| **货币政策** | 美联储声明/点阵图、PBOC政策方向、全球主要央行动作 | 美联储加息/缩表 → 全球紧缩压力；降息/扩表 → 全球流动性宽松 |
| **辅助确认** | 美债10年期收益率、美元指数DXY趋势 | 美元走强+长端利率上行 → 全球资金收紧，股权压力；反之宽松 |

> **全局象限判断的核心逻辑**：美国是全球流动性锚，中国是全球增长的重要变量。两者方向一致时象限判断明确；分化时（如美国滞胀+中国复苏）需综合权重判断，并在快照中记录分化情况。

> **数据要求（硬规则）**：上述GDP/CPI/PMI/收益率等指标必须来自可复现查询的数据源（手工路径 / API / MCP / skill），并在季度快照中记录来源与日期；不得凭”感觉”填数。接口化规范见「数据源提供逻辑.md」。

> **象限信号不明确时**：维持上一次判断，不主动调整配比，等待信号清晰。

### 2.3 市场温度

分两层：**全局温度**调节总股权上限，**分市场温度**约束各市场子上限。

---

#### 2.3a 全局市场温度 → 调节总股权上限

全局温度是对"当前股票资产整体贵不贵"的综合感知，信号来自多市场加权判断。

**判断信号（三类信号同向才算强信号）：**

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **身边信号** | 非投资圈的人主动推荐股票（跨市场普遍） | 专业投资者普遍回避股票 |
| **媒体信号** | 多市场头条充斥"新时代""这次不一样" | 多市场头条充斥"崩盘""史上最差" |
| **全球估值** | A/H/US主要指数PE多数处于历史高分位 | 多数指数PE处于历史低分位 |
| **全球流动性** | 全球信用利差极窄、杠杆资金高、IPO大量 | 信用利差大幅扩大、基金赎回潮 |
| **避险情绪** | VIX<15 且全球风险偏好极高 | VIX>30 或多市场同步恐慌 |

**全局温度 → 总股权基准调节：**

| 全局温度 | 调节方向 | 行动含义 |
|---------|---------|---------|
| 极度狂热 | 基准 −15% 至 −20% | 大量积累弹药，停止追涨 |
| 乐观偏热 | 基准 −5% 至 −10% | 放慢入场节奏，不追高 |
| 冷静正常 | ±0% | 按基准执行 |
| 悲观冷清 | ±0% | 维持，不强行入场，等待机会 |
| 极度恐慌 | 基准 +10% 至 +20% | 动用弹药，接近股权上限 |

---

#### 2.3b 分市场温度 → 各市场股权子上限

A股、港股、美股各自有独立的市场温度，独立判断，彼此不干扰。各市场子上限是软约束：温度冷 → 放宽子上限，好标的可积极买入；温度热 → 压缩子上限，严控该市场总仓位。

**A股温度判断信号：**

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 沪深300/上证PE历史80%+分位 | PE历史20%−分位 |
| **情绪/资金** | 两融余额高位/新开户暴增/公募发行火爆 | 两融余额萎缩/基金大额赎回 |
| **成交量** | 单日成交持续超1.5万亿 | 单日成交持续低于7000亿 |
| **政策情绪** | 利好已充分定价、市场快速拉升后高位震荡 | 政策超预期但市场仍低迷 |

**港股(H)温度判断信号：**

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 恒指/国企指数PE历史80%+分位/AH溢价收窄至低位 | PE历史20%−分位/AH大幅折价 |
| **南向资金** | 南向持续大幅净流入、港股通额度饱和 | 南向持续净流出/外资大幅撤离 |
| **全球风险偏好** | 全球高收益利差极窄/美元走弱 | 高收益利差扩大/美元持续走强 |
| **流动性** | 港股成交量持续放大/杠杆资金高 | 港股成交持续低迷/流动性溢价显著 |

**美股(US)温度判断信号：**

| 维度 | 偏热信号 | 偏冷信号 |
|------|---------|---------|
| **估值** | 标普500/纳斯达克PE历史80%+分位/Cape Shiller高 | PE历史20%−分位 |
| **波动率** | VIX < 13（市场极度自满） | VIX > 30（恐慌） |
| **情绪/资金** | 散户持仓比例极高/AAII看涨极高/IPO大量 | AAII看跌极高/基金大额赎回 |
| **信用市场** | 高收益债利差极窄 | 高收益债利差大幅扩张 |

**分市场温度 → 该市场股权子上限：**

| 该市场温度 | 子上限（占总资产%） | 说明 |
|----------|-----------------|------|
| 极度狂热 | ≤ 10% | 严控，不追涨；已持仓考虑减仓 |
| 乐观偏热 | ≤ 30% | 谨慎，不加仓，好标的小仓试探 |
| 冷静正常 | ≤ 50%（受全局上限约束） | 正常运作，个股机会驱动 |
| 悲观冷清 | 不设独立子上限 | 好标的可积极建仓，机会驱动 |
| 极度恐慌 | 不设独立子上限 | 动用弹药积极入场，接近子上限弹性 |

> **三市场子上限之和通常超过全局股权上限**——这是正常的。子上限是各市场的独立约束，实际持仓由个股机会驱动，三市场合计不超过全局股权上限是最终兜底。

> **子上限参考值非固定**：冷静正常时的50%仅为参考量级，实际由全局上限和当前已用仓位决定。关键逻辑是：越热的市场越压缩上限，越冷的市场越放开约束。

### 2.4 输出：当前目标配比

```
【第一层：总体配比】
第一步：根据全局宏观象限 → 确定总股权基准范围
第二步：根据全局市场温度 → 在基准范围内调节，得到总股权上限
第三步：根据宏观信号 → 确定对冲层（黄金为主，见4.2；SPY PUT按触发条件）
第四步：债券层（当前暂0%，见4.3）
第五步：投机层（加密货币，0-2%，见4.5；可长期维持0%）
第六步：现金层 = 100% − 股权 − 对冲 − 债券 − 投机

【第二层：市场内部约束】
第七步：判断A股/港股/美股各自市场温度 → 确定各市场股权子上限
第八步：更新各行业卡热度检测 → 判断目标行业是否过热/过剩
        🔴过热/泡沫 → 该行业个股仓位额外压缩（甚至回避）
        🟡偏热 → 仓位打七折
        🟢正常/🔵低温 → 不额外限制，按个股分析配置
第九步：在总股权上限 × 市场子上限 × 行业热度折扣的三层约束下，
        由个股定性+定量分析驱动实际买入
        （无固定市场分配比例，哪里有好标的且合理定价且行业不过热就买）
```

**当前环境示例（2026年3月）** [AI演示，数据需自行核实，不作为投资建议]：

```
【第一层：全局判断】

全局宏观象限：
  美国：PMI约在荣枯线附近，通胀有粘性但整体下行，降息周期已启动
  中国：经济复苏节奏偏慢，PPI仍弱，内需承压
  综合：偏"增长+通胀下降"象限，但增长质量偏弱
  → 股权基准：55–65%

全局市场温度：
  美股PE处于历史高分位，AI叙事主导，VIX偏低
  A股/港股估值相对偏低，但全球风险偏好以美股为主导
  综合判断：偏"乐观偏热"（主要拉动来自美股）
  → 调节：基准 −5% 至 −10%
  → 总股权上限：约 48–58%

对冲判断：实际利率仍有粘性，地缘风险持续 → 黄金：7–9%

输出（第一层）：
  股权上限：约 48–58%
  对冲：约 7–10%（黄金为主）
  债券：0%
  现金：约 32–45%

【第二层：分市场温度（示意，数据需自行核实）】

A股温度：悲观冷清  → 子上限：不设（机会驱动）
港股(H)温度：悲观冷清  → 子上限：不设（机会驱动）
美股(US)温度：乐观偏热  → 子上限：≤ 20%

含义：在总股权上限48–58%内，可以在A股/港股积极寻找个股机会；
      美股部分严控在20%以内，不追高。
```

### 2.5 行业/产业链分析层

**定位**：个股买入前的行业级风险过滤层。检测目标行业当前是否存在整体过热、估值泡沫或产能过剩，对后续个股仓位设置额外约束。不替代个股分析，不负责"找到哪里有机会"。

> 核心逻辑：即使个股质量很好，若整个行业处于泡沫化/过剩状态，仍需压缩仓位——因为行业系统性回调会拖累所有成员，基本面优势无法对冲行业整体杀估值的风险。

**在决策流程中的位置**：
```
全局宏观象限 → 总股权上限
分市场温度 → 各市场子上限
行业热度检测 → 对特定行业个股设行业级仓位折扣（过热则压缩，正常则放行）
个股定性+定量 → 实际买入决策
```

**更新节奏**：定性内容（产业链结构、过热信号维度）基本稳定；当前热度判断字段每季度更新一次。

#### 通用行业分析模板

每个行业的分析卡包含两部分：**稳定框架**（不频繁改）+ **当前热度检测**（季度更新）。

```
【稳定框架】
行业性质：____（平台/制造/周期/成长等）
产业链结构：上游____→ 核心环节____→ 下游____
过热/过剩的典型前兆：____（该行业历史上泡沫化前会出现什么信号）
周期类型：____（与宏观强相关/弱周期/政策驱动/技术驱动）
跨市场分布：A股____/ 港股____/ 美股____

【当前热度检测（季度更新）】
```

## 三、再平衡机制

### 3.1 触发规则

| 规则类型 | 触发标准 | 说明 |
|---------|---------|------|
| **时间触发** | 每月检视一次 | 固定日期（如每月第一个周末），同时更新月度快照 |
| **阈值触发** | 股权/对冲/债券/投机四层偏离目标合计 >5个百分点 | 现金层为残余项，无独立目标值，不单独检测；四层合计偏离才触发 |
| **极端触发** | 单一资产层单月涨/跌超20% | 强制再平衡，不等月底 |
| **宏观切换** | 宏观象限发生明确切换 | 重新计算目标配比，按新目标调整 |

### 3.2 操作原则

- 优先用**新增资金**买入低配资产，减少频繁买卖的摩擦成本
- 宏观象限切换引发的调整可分2–3次执行，不必一次到位
- 再平衡机械执行规则，**不受当时市场情绪影响**
- 极度恐慌时超配股权（接近上限），待市场回暖后通过再平衡恢复

### 3.3 月度快照机制

**目的**：防止"完全动态"演变成"完全随意"。每月记录一次，下月对比，发现自己是否在漂移。

### 4.2 对冲层：三类工具

对冲层总上限：**15%**。含两类工具：黄金和SPY PUT，共享这15%额度，各自有独立的纪律规则。

> **加密货币不在对冲层。** 加密货币与股权高度正相关，无法可靠对冲系统性风险，风险特征与黄金/SPY PUT根本不同，单独列为投机层（见4.5）。

#### 4.2.1 黄金（范围：0%–10%）

**定位**：对冲货币贬值、系统性风险和极端地缘事件。不以盈利为目的，视为保险费。

**黄金定价驱动因子：**

| 驱动因子 | 利好黄金（加仓信号） | 利空黄金（减仓信号） |
|---------|--------------------|--------------------|
| **实际利率** | 实际利率下降或为负 | 实际利率大幅上升 |
| **美元走势** | 美元走弱 | 美元持续走强 |
| **通胀预期** | 通胀预期上升 | 通胀预期下降且经济健康 |
| **地缘风险** | 地缘冲突升级、国际秩序不稳定 | 地缘风险缓和 |
| **央行行为** | 全球央行持续购金（去美元化） | 央行抛售黄金 |
| **债务周期** | 主权债务风险上升、财政赤字失控 | 财政状况改善 |

**操作规则：**

| 信号强度 | 判断标准 | 操作 |
|---------|---------|------|
| 强加仓 | ≥3个因子同时利好 | 增配至本象限上限，可一次性调整 |
| 温和加仓 | 1–2个利好因子 | 逐步增配，分2–3次买入 |
| 中性 | 利好利空因子大致平衡 | 维持当前比例 |
| 减仓 | ≥3个利空因子 + 季度涨幅>20% | 减配至本象限下限，部分获利了结 |

**买入工具：**

| 工具 | 适合场景 | 优缺点 |
|------|---------|-------|
| 黄金ETF（如518880、GLD） | 主要配置工具 | 流动性好、费率低 |
| 实物黄金（金条/金币） | 极端避险需求 | 无对手方风险，存储成本高 |
| 纸黄金/黄金账户 | 小额灵活配置 | 操作简单，有银行信用风险 |

---

#### 4.2.2 SPY PUT（范围：年度权利金成本 ≤ 2–3%）

**定位**：尾部风险对冲，不是方向性押注。在市场极度乐观+高估值时买入保护，防范黑天鹅。

**触发买入条件（需同时满足）：**

| 条件 | 标准 |
|------|------|
| 估值信号 | 标普500 PE处于历史90%以上分位 |
| 波动率信号 | VIX < 13（期权便宜） |
| 仓位信号 | 当前股权层处于较高位置 |
| 风险信号 | 存在可识别的系统性风险（如主权债务、地缘升级、政策急转弯） |

> 四个条件需同时满足，缺一不可。不能因为"感觉要跌"就买PUT。

**仓位纪律：**

| 规则 | 具体标准 |
|------|---------|
| 单次仓位上限 | 每笔买入 ≤ 组合总额的 0.5% |
| 年度总上限 | 全年累计权利金支出 ≤ 组合总额的 2–3% |
| 期限选择 | 优先3–6个月到期，不买超短期（Gamma风险）或超长期（资金占用高） |
| 执行价选择 | 标的价格的90–95%（虚值PUT），不买深度实值 |

**卖出/平仓纪律：**

| 情景 | 操作 |
|------|------|
| 浮盈 >200% | 部分获利了结（保留1/3继续持有，防止"卖了就跌"） |
| 距到期 <1个月 | 时间价值损耗加速，主动平仓或展期 |
| 原始判断逻辑已无效 | （如VIX已大幅上升、估值已回落）主动平仓，不要恋战 |
| 市场真的暴跌 | 这是最有价值的时刻——用PUT收益补充子弹，积极入场买股权 |

> **最常见的错误**：把SPY PUT当成赌市场方向的工具，频繁操作。年度权利金成本超过3%意味着你在大量消耗弹药保护，这本身说明持仓风险过高，应该降低股权层。

### 4.4 现金层（0%–95%，残余项）

**现金层 = 100% − 股权层 − 对冲层 − 债券层**

现金层不是配置目标，是其他三层决策的结果。**高现金不是问题，它代表"当前没有找到合理定价的好资产"——这是一种纪律，不是失败。**

> **巴菲特参照**：伯克希尔长期持有大量T-bills，不是因为找不到好公司，而是在等待真正肥美的价格。充足的弹药是行动自由的基础。

**现金工具选择：**

| 工具 | 优点 | 适合场景 |
|------|------|---------|
| 货币市场基金 | 随时赎回，零摩擦 | 日常主要工具 |
| 短期国债逆回购 | 安全性高，收益略高 | 大额且短期不用的部分 |
| 银行活期/短期理财 | 操作方便 | 日常流动性需求 |

---

### 4.5 投机层（0%–2%，当前仅加密货币）

**定位**：无内在价值锚、极高波动的纯投机性资产。与对冲层有本质区别：对冲层资产（黄金/SPY PUT）在股权下跌时倾向于上涨或保值；投机层资产（加密货币）与股权高度正相关，熊市时可能同步暴跌。

**规则：**

- 最高仓位：**总组合的2%**，硬性上限，不得突破
- 亏损容忍：该仓位可以100%亏损，不影响整体组合（这是你持有它的前提）
- 不参与宏观象限/市场温度的分析框架，完全独立判断
- 不视为对冲工具，不计入对冲层15%额度

> **达利欧把比特币列为"反法币资产"，本框架不采用这一定位。** 加密货币与股票高度正相关（非负相关），无内在现金流，无法承担对冲功能；将其视为"投机层"而非"对冲层"更符合其实际风险特征。

**投机层不应该长期维持高仓位**。如果你发现自己持续在2%上限附近，说明对投机资产的兴趣超出了合理边界，应回到股权层寻找质量+价格均满足的标的。

## 五、与个股决策体系的关系

```
全局背景判断（市场温度）
    ↓ 决定当前是积极入场还是积累弹药

资产配置框架（本文档）
    ↓ 确定股权层总仓位上限（不可突破的约束）

个股决策（个人投资决策行动指南.md）
    ↓ 在股权层上限内：选什么、买多贵（五步全流程）

执行 + 季度再平衡
```

> **重要提醒**：个股再好，也不能突破资产配置给的股权上限。超上限意味着你在用超出合理水平的风险押注单一判断——违背"先确保生存"的原则。

## 六、常见误区

| 误区 | 正确做法 |
|------|---------|
| 把配比当固定规则照搬 | 配比是宏观判断的动态输出，每季度重新计算 |
| 把高现金视为"浪费"，总想满仓 | 高现金 = 等待更好价格；弹药是行动自由的基础 |
| 只在"感觉该调"时再平衡 | 机械化规则（季度+阈值触发），不受情绪驱动 |
| 宏观象限不确定时强行判断 | 信号不明确时维持上一次判断，等待清晰信号 |
| 股权层全押个股，不配宽基 | 宽基ETF是底仓，个股是卫星；无底仓的组合脆弱性高 |
| 一次性大额调仓 | 宏观象限切换后分2–3次执行，降低时机判断错误的代价 |
| 用SPY PUT赌方向 | SPY PUT是保险，不是投机；年度成本>3%说明持仓风险过高 |
| 把加密货币当对冲工具 | 加密货币与股权高度正相关，熊市时同步暴跌，不是对冲，是投机；归入投机层，上限2% |
| 投机层超过2% | 硬性上限，超出即降回，不给理由 |
