## 但有一个需要警惕的地方

报告指出：

```
毛利率
2022 53%
2023 61%
2024 69%
2025H1 73%
```

并得出结论：

> 护城河加强

但买方必须警惕：

游戏公司毛利率 **极易误导**。

原因：

```
游戏 = 内容行业
```

成本主要是：

```
研发费用
```

研发是 **费用化**。

所以：

```
毛利率虚高
```

真正要看的指标其实是：

```
研发费用率
```

如果研发费用率很高：

```
真实毛利率 = 低
```

这份报告没有拆。

这是一个缺陷。

# 五、估值模型（最大问题）

报告使用：

**DCF**

这在游戏行业是比较危险的。

因为游戏公司有一个问题：

```
现金流不稳定
```

例如：

```
爆款 → FCF暴涨
冷却 → FCF暴跌
```

所以用 DCF 会导致：

> **过度乐观**

---

## 报告的关键假设

报告假设：

```
FCF0 = 20亿
```

来源：

```
2025 profit 指引
```

但这是一个很危险的假设。

因为：

游戏行业的盈利 **高度周期性**。

更保守的方法应该是：

```
cycle normalized earnings
```

例如：

```
3年平均利润
```

如果按历史平均算：

真实 FCF 可能是：

```
12-15亿
```

而不是 20 亿。

---

## 如果用保守假设

DCF会变成：

```
价值 ≈ 200-230亿
```

而不是：

```
278亿
```

这会导致：

当前股价：

```
71 HKD
```

实际上：
